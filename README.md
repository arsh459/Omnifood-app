## Project Overview

### Key Features

- Mobile first responsive design.
- Internal Navigation between sections.
- Smooth Scrolling through vanilla javaScript.
- Modular and Clean Code.

### Technologies Used

- HTML, CSS, JavaScript
- Git, Netlify
